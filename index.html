Projects

  • Event Scheduler

    This project is a web-based event scheduler. It utilizes HTML, CSS, and JavaScript for the front-end, PHP for server-side scripting, and MySQL for the database. The project is hosted on InfinityFree. It follows coding practices such as using separate files for different functionality, database integration, and user input validation.

  • Weather Forecast

    My Weather App is a single-page weather forecasting application built using PHP, HTML, CSS, and JavaScript. It utilizes the WeatherAPI to fetch weather forecast data based on user input. The application is hosted on InfinityFree and incorporates autocomplete functionality for location search, error handling, and responsive design principles.

  • The Recipe Finder

    The Recipe Finder is a responsive single-page web application built using Flask in Python. It utilizes the Spoonacular API to fetch recipes based on user input. The app employs HTML, CSS, and JavaScript for the frontend, Python for the backend, and follows principles of RESTful API design and modular programming.

  • The Stock App

    The Stock App is a responsive web application built using Python, Flask, HTML, CSS, and JavaScript. It utilizes the yfinance library to fetch stock data based on user input. The app incorporates data analysis and visualization using Pandas and Matplotlib. The design follows responsive principles and leverages Bootstrap for styling.

  • Bullet Hell

    The Bullet Hell Game is a visually appealing, responsive game demonstrating key programming concepts and game design. Players control a character, dodging enemies and shooting bullets. It features collision detection, power-up generation, score tracking, and smooth animation. With intuitive controls, it offers an engaging gaming experience, showcasing solid coding practices.

  • Infinite Runner

    "Infinite Runner" is an HTML5 game with smooth animation, endless running, and obstacle avoidance for a high score. It utilizes object-oriented programming principles, collision detection, and game states. The player can jump multiple times while new obstacles spawn regularly, offering an engaging and challenging gameplay experience.
